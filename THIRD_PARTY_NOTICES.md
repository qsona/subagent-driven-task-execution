## Superpowers

This project was inspired by [obra/superpowers](https://github.com/obra/superpowers),
created by Jesse Vincent. In particular, it draws inspiration from Superpowers'
subagent-driven development workflow: fresh task-focused agents, independent
review, revision after review, and a final whole-work review.

This repository is an independent Japanese-language implementation. Its text,
task model, review rubrics, operating modes, prompts, and Codex metadata were
adapted and expanded for this project's goals. It is not affiliated with or
endorsed by Jesse Vincent, obra, or the Superpowers project.
